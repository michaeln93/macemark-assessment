Current Consumables
Select all classes you currently use. Each class serves a different purpose in sterilization monitoring (ISO 11140-1).
Class 1 — Process Indicator
Autoclave tape or printed indicators on packaging. Shows the item has been exposed to the sterilization process (not that it is sterile).
Class 2 — Bowie-Dick Test
Specific-use indicator for air removal testing in pre-vacuum steam sterilizers. Required as a daily test to verify the vacuum system is working correctly.
Class 4 — Multi-Variable
Reacts to two or more critical sterilization variables (e.g. time and temperature). Used for routine pack/load monitoring.
Class 5 — Integrating Indicator
Reacts to all critical variables of the sterilization process. Performance correlates with biological indicator results — a reliable in-pack monitor.
Class 6 — Emulating Indicator
Cycle-specific indicator that reacts to all critical variables for one specific sterilization cycle type (e.g. 134°C/3.5 min). The most precise chemical indicator.
